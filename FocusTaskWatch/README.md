# 聚焦任务

### 简介

在这个项目中你可以学习以下几种WatchOS开发知识：

1. 发送通知
2. haptic提醒
3. 自定义 Gauge 和 ProgressView
4. 应用数据持久化
5. 基本的 watchOS App 布局
6. TimelineView
7. 本地化
